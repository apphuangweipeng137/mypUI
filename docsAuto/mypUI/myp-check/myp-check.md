| bgType    | 背景主题                       | string  | -      | ''      |
| border    | 边框主题                       | string  | -      | ''      |
| radius    | 圆角主题                       | string  | -      | ''      |
| boxStyle  | 外层样式                       | string  | -      | ''      |
