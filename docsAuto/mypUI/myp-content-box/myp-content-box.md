# myp-content-box

## Props

| Prop name | Description                                   | Type   | Values | Default      |
| --------- | --------------------------------------------- | ------ | ------ | ------------ |
| bgType    | 背景主题                                      | string | -      | 'page'       |
| border    | 边框主题                                      | string | -      | ''           |
| radius    | 圆角主题                                      | string | -      | ''           |
| extra     | 需要从屏幕高度减去的高度                      | string | -      | 'status-nav' |
| height    | height 非 0 时，会直接使用 height，忽略 extra | string | -      | '0'          |
| boxStyle  | 外层样式                                      | string | -      | ''           |
