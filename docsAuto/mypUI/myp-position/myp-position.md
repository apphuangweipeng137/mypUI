# myp-position

## Props

| Prop name | Description     | Type    | Values | Default  |
| --------- | --------------- | ------- | ------ | -------- |
| fixed     | 是否 fixed 定位 | boolean | -      | false    |
| pos       | 定位的位置      | string  | -      | 'bottom' |
| offset    | 定位的偏移量    | string  | -      | '0px'    |
| bgType    | 背景主题        | string  | -      | 'none'   |
| boxStyle  | 外层样式        | string  | -      | ''       |
